What is <!DOCTYPE> ?

Definition: The declaration is not an HTML tag. It is an "information" to the browser about what document type to expect.

Usage:

  • All HTML documents must start with a <!DOCTYPE> declaration.
    Example in HTML 5:
    <!DOCTYPE html>
  • Note: The <!DOCTYPE> declaration is NOT case sensitive.

Tooltip shows up when mouse over by title attribute

Definition: The title attribute specifies extra information about an element.
The information is most often shown as a tooltip text when the mouse moves over the element.

Syntax:
<element title="text">

<b> and <i> VS <strong> and <em>

They have the same effect on normal web browser rendering engines, but there is a fundamental difference between them.

  • <b> and <i> are explicit - they specify bold and italic respectively.
  • <strong> and <em> are semantic - they specify that the enclosed text should be "strong" or "emphasised" in some way, usually bold and italic, but allow for the actual styling to be controlled via CSS. Hence these are preferred in modern web pages.

<bdo> tag to specify the text direction

Definition: BDO stands for Bi-Directional Override.
The <bdo> tag is used to override the current text direction.

Syntax:

  • Text left to right
    <bdo dir="ltr"> โปรแกรมเมอร์ </bdo>
    Result => โปรแกรมเมอร์
  • Text right to left
    <bdo dir="rtl"> โปรแกรมเมอร์ </bdo>
    Result => โปรแกรมเมอร์
