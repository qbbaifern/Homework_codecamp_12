1 CSS syntax ประกอบด้วยอะไรบ้าง
CSS syntax ประกอบด้วย ...
  1. Selector
    • เป็นตัวชี้ไปที่ HTML elements ที่เราต้องการกำหนด style
  2. Declaration block
    • จะประกอบด้วยชุดคำสั่ง(มากกว่า 1 คำสั่งก็ได้) ที่ต้องจบหรือคั่นระหว่างคำสั่งด้วย semi-colon (;)
    • แต่ละชุดคำสั่งประกอบด้วย Property name และ Value และมี semi-colon (;) คั่น
    • ชุดคำสั่งทั้งหมดจะถูกครอบด้วย curly braces ({ })
  3. Property
    • ชื่อของคำสั่งที่ต้องการจัดการ
  4. value
    • ค่าของคำสั่งที่ต้องการ
2 การเขียน CSS มีกี่แบบอะไรบ้าง
การเขียน CSS สามารถทำได้ 3 แบบ คือ ...
  1. Inline CSS
    • กำหนด CSS style ภายใน HTML tag
      EX:
      <p style="color: red;">
  2. Internal CSS
    • กำหนด CSS style ใน head tag
      EX:
      <head>
      <style>
      <p style="color: red;">
      </style>
      </head>
  3. External CSS
    • กำหนด CSS style ใน file .css
    • ที่ file .html จำต้องมีการเรียกใช้ file .css
      EX: สร้าง file ชื่อ "style.css" เพื่อใช้เขียน CSS
      • เรียกใช้ file .css ที่ head tag ของ file .html
        <head>
        <link rel="stylesheet" href="style.css">
        </head>
      • กำหนด style ที่ element ที่ต้องการ ใน file .css ตาม syntax
        p{
           color:red;
        }
3 selector มีกี่แบบอะไรบ้าง
Selector ใน CSS มี 5 ประเภท ได้แก่
  1. Simple selectors(ใช้เลือก elements ตาม name, id, class)
    • สรุปตามตารางด้านล่าง
      Selector Example Example description
      #id #firstname Selects the element with id="firstname"
      .class .intro Selects all elements with class="intro"
      element.class p.intro Selects only

      elements with class="intro"
      * * Selects all elements
      element p Selects all

      elements
      element,element,.. div, p Selects all <div> elements and all <p> elements
  2. Combinator selectors (ใช้เลือก elements ที่มีความสัมพันธ์กัน) มี 4 ประเภท...
    • descendant selector (space)
    • child selector (>)
    • adjacent sibling selector (+)
    • general sibling selector (~)
  3. Pseudo-class selectors (ใช้เลือก elements ที่ต้องการกำหนด state เพิ่ม)
    • Syntax: 
      selector:pseudo-class {
    property: value;
}
    • EX: 
      a:hover {
    color: red;
}
  4. Pseudo-elements selectors (ใช้เลือกและกำหนด stlye ในบางส่วนของ element หรือ content)
    • Syntax: 
      selector::pseudo-element {
    property: value;
}
    • EX: 
      p::first-line {
    color: red;
    font-variant: small-caps;
}
    • สรุปตามตารางด้านล่าง
      Selector Example Example description
      ::after p::after Insert something after the content of each

      element
      ::before p::before Insert something before the content of each

      element
      ::first-letter p::first-letter Selects the first letter of each

      element
      ::first-line p::first-line Selects the first line of each

      element
      ::marker ::marker Selects the markers of list items
      ::selection p::selection Selects the portion of an element that is selected by a user
  5. Attribute selectors (ใช้เลือก elements โดยชี้ไปที่ attribute หรือ attribute value)
    • EX: 
      a[target]{
    background-color: yellow;
}
    • สรุปตามตารางด้านล่าง
      Selector Example Example description
      [attribute] [target] Selects all elements with a target attribute
      [attribute=value] [target=_blank] Selects all elements with target="_blank"
      [attribute~=value] [title~=flower] Selects all elements with a title attribute containing the word "flower"
      [attribute|=value] [lang|=en] Selects all elements with a lang attribute value starting with "en"
      [attribute^=value] a[href^="https"] Selects every element whose href attribute value begins with "https"
      [attribute$=value] a[href$=".pdf"] Selects every element whose href attribute value ends with ".pdf"
      [attribute*=value] a[href*="w3schools"] Selects every element whose href attribute value contains the substring "w3schools"
4 combinator มีกี่แบบอะไรบ้าง
Combinators มี 4 แบบ ดังนี้
  1. descendant selector (space): ชี้ไปที่ element ที่เป็นส่วนสืบทอดจาก element หลักทั้งหมด
    ส่วน css:
    div p {
    background-color: yellow;
    }
    ส่วน html (จะแสดงผลที่บรรทัด highlight): 
    <p>parahraph</p>
<div> 
    <p>parahraph</p>
    <section>
        <p>parahraph</p>
    </section>
</div>
  2. child selector (>): ชี้ไปที่ element ที่เป็นส่วนสืบทอดจาก element โดยตรง
    ส่วน css:
    div > p {
    background-color: yellow;
    }
    ส่วน html (จะแสดงผลที่บรรทัด highlight): 
    <p>parahraph</p>
<div> 
    <p>parahraph</p>
    <section>
        <p>parahraph</p>
    </section>
</div>
  3. adjacent sibling selector (+): ชี้ไปที่ element ถัดจาก element ที่สนใจ 1 element
    ส่วน css:
    div + p {
    background-color: yellow;
    }
    ส่วน html (จะแสดงผลที่บรรทัด highlight): 
    <p>parahraph</p>
<div> 
    <p>parahraph</p>
</div>
<p>parahraph</p>
<p>parahraph</p>
<div> 
    <p>parahraph</p>
</div>
<p>parahraph</p>
  4. general sibling selector (~): ชี้ไปที่ element ถัดจาก element ที่สนใจ ทุกๆ element
    ส่วน css:
    div ~ p {
    background-color: yellow;
    }
    ส่วน html (จะแสดงผลที่บรรทัด highlight): 
    <p>parahraph</p>
<div> 
    <p>parahraph</p>
</div>
<p>parahraph</p>
<p>parahraph</p>
<code>parahraph</code>
<p>parahraph</p>
